    Completely replace `src/app/page.tsx` with a "use client" component implementing the IDKit Selfie Check flow.

    **File structure (top to bottom):**

    1. **Directive and imports:**
       ```
       "use client";
       import { useState } from "react";
       import { IDKitRequestWidget, selfieCheckLegacy } from "@worldcoin/idkit";
       import type { IDKitResult, RpContext } from "@worldcoin/idkit";
       ```

    2. **Constants (module-level, outside component):**
       ```
       const APP_ID = process.env.NEXT_PUBLIC_APP_ID as `app_${string}`;
       const RP_ID = process.env.NEXT_PUBLIC_RP_ID as string;
       const ACTION = "selfie-check";
       ```

    3. **fetchRpContext function (module-level):**
       - `POST /api/rp-signature` with body `{ action }` and header `content-type: application/json`
       - On `!response.ok`: parse JSON, throw `new Error(payload.error ?? "Failed to fetch RP signature")`
       - On success: parse JSON, return `RpContext` object with field mapping:
         - `rp_id: RP_ID`
         - `nonce: data.nonce`
         - `created_at: data.created_at`
         - `expires_at: data.expires_at`
         - `signature: data.sig` (CRITICAL: map `sig` to `signature`)

    4. **verifyProof function (module-level):**
       - `POST /api/verify-proof` with body `{ rp_id: RP_ID, devPortalPayload: payload }` where payload is `IDKitResult`
       - Parse response JSON
       - On `!response.ok`: `throw new Error(json.error ?? "Verification failed")` (CRITICAL: must throw, not return — IDKit treats resolved promise as success)
       - On success: return json

    5. **Home component (default export):**
       - **State (5 useState hooks):**
         - `widgetOpen: boolean` — init `false`
         - `widgetRpContext: RpContext | null` — init `null`
         - `widgetError: string | null` — init `null`
         - `widgetIdkitResult: IDKitResult | null` — init `null`
         - `isLoading: boolean` — init `false`

       - **startWidgetFlow async function:**
         - Clear previous state: `setWidgetError(null)`, `setWidgetIdkitResult(null)`
         - `setIsLoading(true)`
         - try: `const rpContext = await fetchRpContext(ACTION)`
         - Set `setWidgetRpContext(rpContext)` and `setWidgetOpen(true)`
         - catch: `setWidgetError(error instanceof Error ? error.message : "Unknown error")`
         - finally: `setIsLoading(false)`

       - **JSX layout (per UI-SPEC):**
         ```tsx
         <div className="min-h-screen flex flex-col items-center justify-center bg-white dark:bg-zinc-950">
           <div className="flex flex-col items-center gap-4 text-center max-w-sm w-full px-4">
             <h1 className="text-2xl font-semibold">Selfie Test Tester</h1>

             <button
               onClick={startWidgetFlow}
               disabled={isLoading || widgetOpen}
               className="min-h-[44px] py-2 px-6 rounded-full bg-zinc-900 text-white text-sm font-semibold transition-colors hover:bg-zinc-700 disabled:opacity-50 disabled:cursor-not-allowed dark:bg-white dark:text-zinc-900 dark:hover:bg-zinc-200"
             >
               {isLoading ? "Loading..." : "Start Verification"}
             </button>

             {widgetIdkitResult && (
               <p className="text-green-600 text-base">Verified!</p>
             )}

             {widgetError && (
               <p className="text-red-600 text-base">
                 Verification failed{" "}
                 <code className="font-mono text-sm">{widgetError}</code>
               </p>
             )}

             {widgetRpContext && (
               <IDKitRequestWidget
                 open={widgetOpen}
                 onOpenChange={setWidgetOpen}
                 app_id={APP_ID}
                 action={ACTION}
                 rp_context={widgetRpContext}
                 allow_legacy_proofs={true}
                 preset={selfieCheckLegacy({ signal: "selfie-test" })}
                 handleVerify={async (result) => {
                   await verifyProof(result);
                 }}
                 onSuccess={(result) => {
                   setWidgetIdkitResult(result);
                 }}
                 onError={(errorCode) => {
                   setWidgetError(`Verification failed: ${errorCode}`);
                 }}
               />
             )}
           </div>
         </div>
         ```

    **Anti-patterns to avoid:**
    - Do NOT use `next/dynamic` with `{ ssr: false }` — "use client" is sufficient
    - Do NOT fetch rp_context in useEffect on mount — fetch on button click only (prevents expiry)
    - Do NOT resolve handleVerify on failure — must throw to signal IDKit
    - Do NOT render IDKitRequestWidget unconditionally — rp_context is required prop
    - Do NOT use `allow_legacy_proofs: false` with selfieCheckLegacy — must be `true`
    - Do NOT use `data.signature` when reading API response — the field is `data.sig`
